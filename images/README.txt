INSTRUCCIONES PARA IMÁGENES:

1. Coloca tus imágenes en esta carpeta
2. Formatos soportados: JPG, PNG, GIF
3. Tamaño máximo: 5MB por imagen
4. Nombres sin espacios ni caracteres especiales

IMPORTANTE:
- Solo se enviarán imágenes que estén físicamente en esta carpeta
- Si la imagen no existe, el envío fallará
- Para usar imágenes, súbelas a un servicio público (Imgur, etc.) y usa la URL en el CSV
